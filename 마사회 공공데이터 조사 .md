# 한국마사회 공공데이터 API 종합 가이드

한국마사회(Korea Racing Authority, KRA)는 공공데이터포털(data.go.kr)에서 **50개 이상의 Open API**를 제공하며, 경마 정보 서비스 구축에 필수적인 데이터를 포괄합니다. 기관코드 **B551015**로 모든 API 엔드포인트가 `http://apis.data.go.kr/B551015/`로 시작하며, 대부분 자동승인으로 즉시 활용 가능합니다. 일일 호출 제한은 개발계정 기준 **10,000건**이고, 활용사례 등록 시 증가 신청이 가능합니다.

---

- `ym_fr`/`ym_to`: 경주연월 범위 (YYYYMM)
- `pageNo`, `numOfRows`: 페이징 처리

## API 활용 공통 사항

### 인증 및 요청 형식
```
GET http://apis.data.go.kr/B551015/{API코드}/{오퍼레이션}
    ?ServiceKey={인증키}
    &pageNo=1
    &numOfRows=10
    &meet=1
```

### 경마장 코드
| 코드 | 경마장 |
|------|--------|
| 1 | 서울경마장 (과천) |
| 2 | 제주경마장 |
| 3 | 부산경남경마장 (부경) |

### 호출 제한 및 승인
- **개발계정**: 일 10,000건, 자동승인
- **운영계정**: 활용사례 등록 시 트래픽 증가 신청 가능
- **일부 API**: 운영단계 심의승인 필요 (경주마명단, 조교사별두수 등)

### 데이터 갱신
- 대부분의 API는 **날짜 파라미터 미입력 시 최근 1개월** 데이터만 반환
- 경주 당일 API는 실시간 또는 당일 갱신
