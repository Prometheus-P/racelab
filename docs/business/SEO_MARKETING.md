## 2. 기술적 SEO

### 2.1 사이트맵

```xml
<!-- public/sitemap.xml -->
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://krace.co.kr/</loc>
    <changefreq>hourly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://krace.co.kr/horse</loc>
    <changefreq>hourly</changefreq>
    <priority>0.8</priority>
  </url>
  <!-- 동적 생성 필요 -->
</urlset>
```

### 2.2 robots.txt

```txt
# public/robots.txt
User-agent: *
Allow: /

Sitemap: https://krace.co.kr/sitemap.xml

# 관리자/API 차단
Disallow: /api/
```

### 2.3 Core Web Vitals 목표

| 지표 | 목표 | 현재 | 개선 방안 |

## 8. 경쟁사 분석

### 8.1 주요 경쟁사

| 경쟁사 | 강점 | 약점 | 우리의 차별화 |
|--------|------|------|--------------|
| 한국마사회 | 공식 데이터 | UX 구식 | 모던 UI |
| 스피드온 | 종합 플랫폼 | 복잡함 | 심플함 |

### 8.2 차별화 포인트

1. **통합 플랫폼**: 경마+경륜+경정 원스톱
2. **모바일 퍼스트**: 반응형 최적화
3. **빠른 속도**: Next.js SSR
4. **심플 UX**: 불필요한 기능 제거
