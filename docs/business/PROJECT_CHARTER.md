## 2. 비즈니스 목표

### 2.1 문제 정의

- 경마, 경륜, 경정 정보가 각각 다른 사이트에 분산되어 있음
- 기존 사이트들의 UX가 구식이고 모바일 최적화 부족
- 실시간 배당률 확인을 위해 여러 사이트 순회 필요

### 2.2 솔루션

- 경마/경륜/경정 정보를 단일 플랫폼에서 통합 제공
- 모바일 퍼스트 반응형 디자인
- 실시간 데이터 업데이트 (1분 주기)

### 2.3 목표 KPI

| KPI           | 3개월 | 6개월  | 12개월 |
| ------------- | ----- | ------ | ------ |
| MAU           | 5,000 | 20,000 | 50,000 |
| 일일 방문자   | 500   | 2,000  | 5,000  |
| 페이지뷰/방문 | 4     | 5      | 6      |
| 이탈률        | <60%  | <50%   | <45%   |
| 월 수익       | ₩500K | ₩2M    | ₩5M    |

## 3. 수익 모델

> **우선순위**: 제휴 마케팅 (Priority 1) → 광고 수익 (Priority 2) → 프리미엄 서비스

### 3.1 Priority 1: 제휴 마케팅 (Affiliate)

| 제휴처       | 모델 | 예상 단가         | 월 예상 전환 (MAU 10K, 1.5% 전환) | 월 예상 수익    |
| ------------ | ---- | ----------------- | ---------------------------------- | --------------- |
| 스피드온     | CPA  | ₩5,000~10,000/건  | 120건                              | ₩600K~1.2M      |
| 스포츠/금융  | CPA  | ₩10,000~30,000/건 | 30건                               | ₩300K~900K      |
| 파트너 앱    | CPI  | ₩3,000~5,000/설치 | 80건                               | ₩240K~400K      |

- KPI: **제휴 전환률 ≥1.5%**, LTV 기반 파트너 확대
- 운영 방식: 결과 페이지/가이드에 가입 유도 카드, 주간 리포트에 CTA 배치

### 3.2 Priority 2: 광고 수익 (AdSense/애드포스트)

| MAU | 추정 PV/월 | 보수적 RPM | 월 예상 수익 |
| --- | ---------- | ----------- | ------------ |
| 10,000 | ~150,000 | ₩1,300     | **₩200K**    |
| 30,000 | ~450,000 | ₩1,600     | ₩700K        |

- 광고는 UX를 해치지 않는 선에서 상·중·하 배너 3개만 운영
- 목표: **제휴 매출 대비 30% 이하 비중** 유지

### 3.3 Phase 3: 프리미엄 서비스

- AI 분석 데이터 구독: ₩9,900/월 (경주 요약 인사이트, CSV Export)
- 광고 제거 구독: ₩4,900/월 (UX 개선)

## 4. 타겟 사용자

### 4.1 주요 타겟

- **연령대**: 30~60대 남성
- **관심사**: 경마, 경륜, 경정, 스포츠 베팅
- **특성**: 모바일 사용 선호, 실시간 정보 필요

### 4.2 사용자 페르소나

**페르소나 A: 주말 경마 팬 (김철수, 45세)**

- 매주 토/일 경마 시청
- 스마트폰으로 배당률 확인
- Pain Point: 여러 앱 번갈아 사용 불편

**페르소나 B: 경륜 동호회원 (박영호, 52세)**

- 광명경륜장 정기 방문
- 선수 정보, 최근 성적 분석 중시
- Pain Point: 데이터 분석 도구 부족

## 5. 경쟁 분석

| 경쟁사          | 장점        | 단점            |
| --------------- | ----------- | --------------- |
| 한국마사회 공식 | 공식 데이터 | UX 구식, 경마만 |
| 스피드온        | 종합 플랫폼 | 복잡한 UI       |
| 경륜경정정보    | 전문 정보   | 모바일 미지원   |

### 5.1 차별화 전략

1. 통합 플랫폼 (경마+경륜+경정)
2. 모바일 퍼스트 UX
3. 빠른 로딩 속도 (Next.js SSR)
4. 심플하고 직관적인 UI
